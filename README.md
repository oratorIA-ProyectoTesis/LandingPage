# OratorIA — Landing page

Proyecto **independiente** de la plataforma (deploy y ciclo de vida propios).

- **Stack:** React 19 + TypeScript + Vite + Tailwind v4.
- **Marca:** tokens "Spotlight" en `src/index.css` (fuente de verdad:
  vault `09 - Producto/Branding.md`). Fuentes: Baloo 2 · Hanken Grotesk · Fraunces.
- **Deploy:** Vercel Hobby.
- **Referencia visual:** `TESIS_ORATORIA/DESING/oratoria-brandboard.html`
  (hero, botón chunky 3D, mascota Vox, La Sala).

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # producción
```
